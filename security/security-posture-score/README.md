# Security Posture Score

Cross-domain security posture scoring — aggregates findings, metrics, and control coverage into a 0–100 executive scorecard.

## When to use

- Preparing for a quarterly board security briefing
- Tracking security program improvement over time
- Benchmarking posture against prior periods
- Communicating overall security health to non-technical executives

## Quick Start

```bash
python scripts/security-posture-score_tool.py --help
python scripts/security-posture-score_tool.py --output json
```

## Skill Level: L3

Produces advisory output only. No mutating actions. Safe to run without human approval gate.
